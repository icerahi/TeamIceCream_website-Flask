Flask Web Development

Flask is a Python micro-framework which does not require any particular third party tools or components. While many extensions exist for common tasks such as object-relational mapping, form validation and upload handling, Flask itself is lightweight and highly customisable. Its small and non-intrusive nature makes it ideal for niche apps, and its track record includes Pinterest and LinkedIn.

Django Web Development

Django is an open-source high-level Python web framework that focuses on rapid development and clear design. Using Django simplifies the process of creating a complex, database-driven web application, providing optional modules such as authentication and CRUD, while keeping in line with a consistent Python philosophy. Any such application will be scalable and versatile, with successful Django projects ranging from CMS platforms to scientific computing platforms.
